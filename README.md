# cw-official-docs — Canonical wiki pages for your fictional world

> Create polished, citable reference documentation for story characters, locations, lore, and events. The single source of truth for your fictional world.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
cw-official-docs produces encyclopedic wiki-style pages from confirmed story facts. Unlike brainstorming notes, documentation here is finalized and polished — no `[TBD]` markers, no exploratory options. Every claim requires a citation (chapter reference or worldbuilding document). The tone is third-person, neutral, and encyclopedic, making pages suitable for sharing with collaborators or building a reader-facing wiki.

## Features
- **Canonical only** — confirmed facts from written chapters or finalized worldbuilding
- **Citations required** — every claim sourced to a chapter number or document
- **Encyclopedic tone** — 3rd person, neutral, past tense for events
- **Document types** — character profiles, location docs, lore wikis, event records
- **Key distinction** — documentation = single finalized version; brainstorm = exploratory with options

## Usage / Quick Start
Trigger after finalizing decisions about a character, place, or story element. Provide confirmed facts and source references; the skill formats them into a clean wiki page.
